## Install 

just Run install

```
cd /tmp && wget https://github.com/kooshakoosha/what_is_Kernel/archive/refs/heads/main.zip && unzip main.zip && cd whKernel-main && sudo cp whKernel /usr/bin && cd
```




## Uninstall 

```
sudo rm /usr/bin/whKernel
```
