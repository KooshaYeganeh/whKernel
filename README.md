## Install 

just Run install

```
cd /opt 
```
```
wget https://github.com/kooshakoosha/what_is_Kernel/archive/refs/heads/main.zip 
```
```
unzip main.zip
```
```
cd whKernel-main/
```
```
sudo cp whKernel /usr/bin
```
```
cd
```




## Uninstall 

```
cd /opt && sudo rm main.zip && sudo rm -rf what_is_Kernel-main && sudo rm /usr/bin/whKernel
```
